Knock integrates with [Vonage](https://vonage.com/) to send SMS notifications to your recipients.

## Features

- Per environment configuration
- Sandbox mode

## Getting started

You can create a new Vonage channel in the dashboard under the "Channels" section. From there, you'll need to configure the channel for each environment you have.

## Provider configuration

  />

## Recipient data requirements

In order to send an SMS notification you'll need a valid `phone_number` property set on your recipient.
