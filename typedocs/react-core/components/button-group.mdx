Used to group and space multiple `Button` components into a single line.

## Usage

```jsx
import { Button, ButtonGroup } from "@knocklabs/react-core";

function MyComponent() {
  return (
    <ButtonGroup>
      <Button onClick={() => console.log("Action 1")}>Action 1</Button>
      <Button variant="secondary" onClick={() => console.log("Action 2")}>
        Action 2
      </Button>
    </ButtonGroup>
  );
}
```

## Props

<Attributes>
  <Attribute
    name="children"
    type="ReactNode"
    description="One or more `Button` components"
  />
</Attributes>
