Represents a Microsoft Teams channel with its properties.

## Properties

<Attributes>
  <Attribute
    name="id"
    type="string"
    description="The unique identifier of the Microsoft Teams channel."
  />
  <Attribute
    name="displayName"
    type="string"
    description="The display name of the channel."
  />
  <Attribute
    name="description"
    type="string"
    description="Optional description of the channel."
  />
  <Attribute
    name="membershipType"
    type="string"
    description="Optional membership type of the channel."
  />
  <Attribute
    name="isArchived"
    type="boolean"
    description="Optional flag indicating if the channel is archived."
  />
  <Attribute
    name="createdDateTime"
    type="string"
    description="Optional creation date and time of the channel."
  />
</Attributes>

## Example

```typescript
const channel: MsTeamsChannel = {
  id: "19:abc123def456...",
  displayName: "General",
  description: "General discussion channel",
  membershipType: "standard",
  isArchived: false,
  createdDateTime: "2023-01-15T10:30:00Z",
};
```
