Configuration object for internationalization content used to set translations available in child components exposed under `KnockFeedProvider`, `KnockSlackProvider`, and `KnockMsTeamsProvider`.

## Properties

<Attributes>
  <Attribute
    name="translations"
    type="Partial&lt;Translations&gt;"
    description="A partial object containing translation overrides for the default English strings."
  />
  <Attribute
    name="locale"
    type="string"
    description="A valid locale code (e.g., 'en', 'es', 'fr')."
  />
</Attributes>

## Translations Interface

The `Translations` interface includes the following properties:

<Attributes>
  <Attribute
    name="emptyFeedTitle"
    type="string"
    description="Title shown when the feed is empty."
  />
  <Attribute
    name="emptyFeedBody"
    type="string"
    description="Body text shown when the feed is empty."
  />
  <Attribute
    name="notifications"
    type="string"
    description="Label for notifications."
  />
  <Attribute name="poweredBy" type="string" description="Powered by text." />
  <Attribute
    name="markAllAsRead"
    type="string"
    description="Mark all as read button text."
  />
  <Attribute
    name="archiveNotification"
    type="string"
    description="Archive notification button text."
  />
  <Attribute name="all" type="string" description="All filter option text." />
  <Attribute
    name="unread"
    type="string"
    description="Unread filter option text."
  />
  <Attribute name="read" type="string" description="Read filter option text." />
  <Attribute
    name="unseen"
    type="string"
    description="Unseen filter option text."
  />
</Attributes>
