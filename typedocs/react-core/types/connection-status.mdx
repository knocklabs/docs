Represents the current status of a connection to an external service like Slack or Microsoft Teams.

## Values

<Attributes>
  <Attribute
    name="connecting"
    type="string"
    description="Connection is being established."
  />
  <Attribute
    name="connected"
    type="string"
    description="Connection has been successfully established."
  />
  <Attribute
    name="disconnected"
    type="string"
    description="No connection exists."
  />
  <Attribute
    name="error"
    type="string"
    description="Connection failed or encountered an error."
  />
  <Attribute
    name="disconnecting"
    type="string"
    description="Connection is being terminated."
  />
</Attributes>

## Example

```typescript
const [status, setStatus] = useState<ConnectionStatus>("disconnected");

// When starting connection
setStatus("connecting");

// On successful connection
setStatus("connected");

// On error
setStatus("error");
```
