Configuration options for querying Slack channels.

## Properties

<Attributes>
  <Attribute
    name="maxCount"
    type="number"
    description="The maximum number of channels to return. Default: 1000."
  />
  <Attribute
    name="limitPerPage"
    type="number"
    description="How many Slack channels will be returned per request. Default: 200."
  />
  <Attribute
    name="excludeArchived"
    type="boolean"
    description="Whether to exclude archived channels. Default: true."
  />
  <Attribute
    name="types"
    type="string"
    description="Types of channels to return. Default: 'private_channel,public_channel'."
  />
  <Attribute
    name="teamId"
    type="string"
    description="Filters channels to a specific team ID. Default: null."
  />
</Attributes>

## Example

```typescript
const queryOptions: SlackChannelQueryOptions = {
  maxCount: 500,
  limitPerPage: 100,
  excludeArchived: true,
  types: "public_channel",
  teamId: "T1234567890",
};
```
