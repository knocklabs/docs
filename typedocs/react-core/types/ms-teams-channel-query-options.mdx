Configuration options for querying Microsoft Teams channels within a team.

## Properties

<Attributes>
  <Attribute
    name="filter"
    type="string"
    description="OData $filter query parameter to filter channels. Default: 'isArchived eq false and membershipType eq 'standard''."
  />
  <Attribute
    name="select"
    type="string"
    description="OData $select query parameter to select fields. Default: 'id,displayName'."
  />
</Attributes>

## Example

```typescript
const queryOptions: MsTeamsChannelQueryOptions = {
  filter: "isArchived eq false and membershipType eq 'standard'",
  select: "id,displayName,description,membershipType",
};
```
