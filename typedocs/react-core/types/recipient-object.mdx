Represents a recipient object used to identify a specific object and collection in Knock.

## Properties

<Attributes>
  <Attribute
    name="objectId"
    type="string"
    description="The unique identifier of the object."
  />
  <Attribute
    name="collection"
    type="string"
    description="The collection name that the object belongs to."
  />
</Attributes>

## Example

```typescript
const recipientObject: RecipientObject = {
  objectId: "team-123",
  collection: "teams",
};

// Used with Slack channels
const slackRecipient: RecipientObject = {
  objectId: "workspace-456",
  collection: "workspaces",
};
```
