The `useSlackAuth` hook is used to manage Slack authentication for connecting user accounts to Slack workspaces.

## Parameters

This hook does not accept any parameters.

## Returns

A `UseSlackAuthReturn` object with the following properties:

<Attributes>
  <Attribute
    name="authenticate"
    type="(options?: SlackAuthOptions) => Promise<void>"
    description="Function to initiate Slack authentication flow."
  />
  <Attribute
    name="revokeAccess"
    type="() => Promise<void>"
    description="Function to revoke Slack access for the current user."
  />
  <Attribute
    name="isLoading"
    type="boolean"
    description="Whether an authentication operation is in progress."
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useSlackAuth` hook to manage Slack authentication.

```tsx
import { useSlackAuth } from "@knocklabs/react";

const MyComponent = () => {
  const { authenticate, revokeAccess, isLoading } = useSlackAuth();

  const handleConnect = async () => {
    await authenticate();
  };

  const handleDisconnect = async () => {
    await revokeAccess();
  };

  return (
    <div>
      <button onClick={handleConnect} disabled={isLoading}>
        Connect Slack
      </button>
      <button onClick={handleDisconnect} disabled={isLoading}>
        Disconnect Slack
      </button>
    </div>
  );
};
```
