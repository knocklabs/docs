The `useConnectedSlackChannels` hook is used to retrieve Slack channels that are currently connected for the user.

## Parameters

This hook does not accept any parameters.

## Returns

A `UseConnectedSlackChannelsReturn` object with the following properties:

<Attributes>
  <Attribute
    name="channels"
    type="ConnectedSlackChannel[]"
    description="Array of connected Slack channels"
  />
  <Attribute
    name="isLoading"
    type="boolean"
    description="Whether connected channels are being loaded"
  />
  <Attribute
    name="error"
    type="Error | null"
    description="Error object if the request failed"
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useConnectedSlackChannels` hook to display connected Slack channels.

```tsx
import { useConnectedSlackChannels } from "@knocklabs/react";

const MyComponent = () => {
  const { channels, isLoading, error } = useConnectedSlackChannels();

  if (isLoading) return <div>Loading connected channels...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <h2>Connected Channels</h2>
      <ul>
        {channels.map((channel) => (
          <li key={channel.id}>
            {channel.name} - {channel.workspace_name}
          </li>
        ))}
      </ul>
    </div>
  );
};
```
