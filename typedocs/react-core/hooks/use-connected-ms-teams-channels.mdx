The `useConnectedMsTeamsChannels` hook is used to retrieve Microsoft Teams channels that are currently connected for the user.

## Parameters

This hook does not accept any parameters.

## Returns

A `UseConnectedMsTeamsChannelsReturn` object with the following properties:

<Attributes>
  <Attribute
    name="channels"
    type="ConnectedMsTeamsChannel[]"
    description="Array of connected Microsoft Teams channels."
  />
  <Attribute
    name="isLoading"
    type="boolean"
    description="Whether connected channels are being loaded."
  />
  <Attribute
    name="error"
    type="Error | null"
    description="Error object if the request failed."
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useConnectedMsTeamsChannels` hook to display connected Microsoft Teams channels.

```tsx
import { useConnectedMsTeamsChannels } from "@knocklabs/react";

const MyComponent = () => {
  const { channels, isLoading, error } = useConnectedMsTeamsChannels();

  if (isLoading) return <div>Loading connected channels...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <h2>Connected Channels</h2>
      <ul>
        {channels.map((channel) => (
          <li key={channel.id}>
            {channel.displayName} - {channel.teamName}
          </li>
        ))}
      </ul>
    </div>
  );
};
```
