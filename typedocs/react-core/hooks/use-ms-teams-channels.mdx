The `useMsTeamsChannels` hook is used to retrieve available Microsoft Teams channels for a specific team.

## Parameters

Accepts an object (`UseMsTeamsChannelsParams`) with the following properties:

<Attributes>
  <Attribute
    name="teamId"
    type="string"
    description="The ID of the Microsoft Teams team to get channels for"
  />
</Attributes>

## Returns

A `UseMsTeamsChannelsReturn` object with the following properties:

<Attributes>
  <Attribute
    name="channels"
    type="MsTeamsChannel[]"
    description="Array of available Microsoft Teams channels"
  />
  <Attribute
    name="isLoading"
    type="boolean"
    description="Whether channels are being loaded"
  />
  <Attribute
    name="error"
    type="Error | null"
    description="Error object if the request failed"
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useMsTeamsChannels` hook to display available Microsoft Teams channels.

```tsx
import { useMsTeamsChannels } from "@knocklabs/react";

const MyComponent = ({ teamId }: { teamId: string }) => {
  const { channels, isLoading, error } = useMsTeamsChannels({ teamId });

  if (isLoading) return <div>Loading channels...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <h2>Available Channels</h2>
      <ul>
        {channels.map((channel) => (
          <li key={channel.id}>{channel.displayName}</li>
        ))}
      </ul>
    </div>
  );
};
```
