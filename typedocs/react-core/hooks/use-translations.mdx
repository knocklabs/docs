The `useTranslations` hook is used to retrieve translations for the current user's locale. It provides access to localized strings for the Knock in-app UI components.

## Parameters

This hook does not accept any parameters.

## Returns

A `UseTranslationsReturn` object with the following properties:

<Attributes>
  <Attribute
    name="t"
    type="(key: string, defaultValue?: string) => string"
    description="Translation function that returns the translated string for the given key."
  />
  <Attribute
    name="locale"
    type="string"
    description="The current locale being used."
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useTranslations` hook to get translated strings.

```tsx
import { useTranslations } from "@knocklabs/react";

const MyComponent = () => {
  const { t, locale } = useTranslations();

  return (
    <div>
      <h1>{t("notification_feed.title", "Notifications")}</h1>
      <p>{t("notification_feed.empty", "No notifications")}</p>
      <small>Current locale: {locale}</small>
    </div>
  );
};
```
