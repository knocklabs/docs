The `useMsTeamsConnectionStatus` hook is used to check and manage the current Microsoft Teams connection status for a user. This hook is typically used internally by the `KnockMsTeamsProvider` and is not commonly used directly in application code.

## Parameters

<Attributes>
  <Attribute
    name="knock"
    type="Knock"
    description="The authenticated Knock client instance."
    isRequired
  />
  <Attribute
    name="knockMsTeamsChannelId"
    type="string"
    description="The Knock channel ID for the Microsoft Teams integration."
    isRequired
  />
  <Attribute
    name="tenantId"
    type="string"
    description="The tenant ID for multi-tenant applications."
    isRequired
  />
</Attributes>

## Returns

Returns an object with the following properties:

<Attributes>
  <Attribute
    name="connectionStatus"
    type="ConnectionStatus"
    description="The current connection status: 'connecting' | 'connected' | 'disconnected' | 'error' | 'disconnecting'."
  />
  <Attribute
    name="setConnectionStatus"
    type="(status: ConnectionStatus) => void"
    description="Function to update the connection status."
  />
  <Attribute
    name="errorLabel"
    type="string | null"
    description="Error message if the connection check failed."
  />
  <Attribute
    name="setErrorLabel"
    type="(errorLabel: string) => void"
    description="Function to set an error message."
  />
  <Attribute
    name="actionLabel"
    type="string | null"
    description="Label for the current action being performed."
  />
  <Attribute
    name="setActionLabel"
    type="(actionLabel: string | null) => void"
    description="Function to set an action label."
  />
</Attributes>

## Example

### Basic usage

This hook is typically used internally by the `KnockMsTeamsProvider`. If you need to access the connection status in your components, use the `useKnockMsTeamsClient` hook instead:

```tsx
import { KnockMsTeamsProvider, useKnockMsTeamsClient } from "@knocklabs/react";

const MsTeamsStatus = () => {
  const { connectionStatus } = useKnockMsTeamsClient();

  return (
    <div>
      <p>Microsoft Teams status: {connectionStatus}</p>
    </div>
  );
};

const App = () => (
  <KnockMsTeamsProvider
    knockMsTeamsChannelId="ms-teams-channel-id"
    tenantId="tenant-id"
  >
    <MsTeamsStatus />
  </KnockMsTeamsProvider>
);
```

### Advanced usage

If you need to use this hook directly:

```tsx
import { useKnockClient, useMsTeamsConnectionStatus } from "@knocklabs/react";

const MyComponent = () => {
  const knock = useKnockClient();
  const { connectionStatus, errorLabel } = useMsTeamsConnectionStatus(
    knock,
    "ms-teams-channel-id",
    "tenant-id",
  );

  if (connectionStatus === "connecting")
    return <div>Checking connection...</div>;
  if (errorLabel) return <div>Error: {errorLabel}</div>;

  return (
    <div>
      <p>Status: {connectionStatus}</p>
    </div>
  );
};
```
