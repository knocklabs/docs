The `useMsTeamsConnectionStatus` hook is used to check the current Microsoft Teams connection status for a user.

## Parameters

This hook does not accept any parameters.

## Returns

A `UseMsTeamsConnectionStatusReturn` object with the following properties:

<Attributes>
  <Attribute
    name="isConnected"
    type="boolean"
    description="Whether the user has a connected Microsoft Teams account"
  />
  <Attribute
    name="connectionStatus"
    type="'connected' | 'disconnected' | 'loading'"
    description="The current connection status"
  />
  <Attribute
    name="error"
    type="Error | null"
    description="Error object if the status check failed"
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useMsTeamsConnectionStatus` hook to check Microsoft Teams connection status.

```tsx
import { useMsTeamsConnectionStatus } from "@knocklabs/react";

const MyComponent = () => {
  const { isConnected, connectionStatus, error } = useMsTeamsConnectionStatus();

  if (connectionStatus === "loading") return <div>Checking connection...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <p>Teams Status: {isConnected ? "Connected" : "Not Connected"}</p>
      {isConnected && <p>You can receive Microsoft Teams notifications</p>}
    </div>
  );
};
```
