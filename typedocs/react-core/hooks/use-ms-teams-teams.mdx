The `useMsTeamsTeams` hook is used to retrieve available Microsoft Teams for a connected workspace.

## Parameters

This hook does not accept any parameters.

## Returns

A `UseMsTeamsTeamsReturn` object with the following properties:

<Attributes>
  <Attribute
    name="teams"
    type="MsTeamsTeam[]"
    description="Array of available Microsoft Teams."
  />
  <Attribute
    name="isLoading"
    type="boolean"
    description="Whether teams are being loaded."
  />
  <Attribute
    name="error"
    type="Error | null"
    description="Error object if the request failed."
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useMsTeamsTeams` hook to display available Microsoft Teams.

```tsx
import { useMsTeamsTeams } from "@knocklabs/react";

const MyComponent = () => {
  const { teams, isLoading, error } = useMsTeamsTeams();

  if (isLoading) return <div>Loading teams...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <h2>Available Teams</h2>
      <ul>
        {teams.map((team) => (
          <li key={team.id}>{team.displayName}</li>
        ))}
      </ul>
    </div>
  );
};
```
