The `useMsTeamsTeams` hook is used to retrieve available Microsoft Teams for a connected workspace. This hook must be used within a `KnockMsTeamsProvider`.

## Parameters

Accepts an object with the following properties:

<Attributes>
  <Attribute
    name="queryOptions"
    type="MsTeamsTeamQueryOptions"
    description="Optional query options for filtering, pagination, and field selection."
  />
</Attributes>

## Returns

Returns an object with the following properties:

<Attributes>
  <Attribute
    name="data"
    type="MsTeamsTeam[]"
    description="Array of available Microsoft Teams."
  />
  <Attribute
    name="isLoading"
    type="boolean"
    description="Whether teams are being loaded or revalidating."
  />
  <Attribute
    name="refetch"
    type="() => void"
    description="Function to manually refetch the teams."
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useMsTeamsTeams` hook to display available Microsoft Teams.

```tsx
import { KnockMsTeamsProvider, useMsTeamsTeams } from "@knocklabs/react";

const MsTeamsTeamList = () => {
  const { data: teams, isLoading, refetch } = useMsTeamsTeams({});

  if (isLoading) return <div>Loading teams...</div>;

  return (
    <div>
      <h2>Available teams</h2>
      <button onClick={refetch}>Refresh</button>
      <ul>
        {teams.map((team) => (
          <li key={team.id}>{team.displayName}</li>
        ))}
      </ul>
    </div>
  );
};

const App = () => (
  <KnockMsTeamsProvider
    knockMsTeamsChannelId="ms-teams-channel-id"
    tenantId="tenant-id"
  >
    <MsTeamsTeamList />
  </KnockMsTeamsProvider>
);
```

### With query options

You can customize the query with pagination and filtering options:

```tsx
const { data: teams } = useMsTeamsTeams({
  queryOptions: {
    maxCount: 500,
    limitPerPage: 50,
    filter: "displayName eq 'Engineering'",
    select: "id,displayName,description",
  },
});
```
