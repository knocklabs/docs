The `useSlackChannels` hook is used to retrieve available Slack channels for a connected workspace.

## Parameters

This hook does not accept any parameters.

## Returns

A `UseSlackChannelsReturn` object with the following properties:

<Attributes>
  <Attribute
    name="channels"
    type="SlackChannel[]"
    description="Array of available Slack channels"
  />
  <Attribute
    name="isLoading"
    type="boolean"
    description="Whether channels are being loaded"
  />
  <Attribute
    name="error"
    type="Error | null"
    description="Error object if the request failed"
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useSlackChannels` hook to display available Slack channels.

```tsx
import { useSlackChannels } from "@knocklabs/react";

const MyComponent = () => {
  const { channels, isLoading, error } = useSlackChannels();

  if (isLoading) return <div>Loading channels...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <h2>Available Channels</h2>
      <ul>
        {channels.map((channel) => (
          <li key={channel.id}>{channel.name}</li>
        ))}
      </ul>
    </div>
  );
};
```
