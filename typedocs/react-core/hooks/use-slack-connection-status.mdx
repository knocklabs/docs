The `useSlackConnectionStatus` hook is used to check the current Slack connection status for a user.

## Parameters

This hook does not accept any parameters.

## Returns

A `UseSlackConnectionStatusReturn` object with the following properties:

<Attributes>
  <Attribute
    name="isConnected"
    type="boolean"
    description="Whether the user has a connected Slack account"
  />
  <Attribute
    name="connectionStatus"
    type="'connected' | 'disconnected' | 'loading'"
    description="The current connection status"
  />
  <Attribute
    name="error"
    type="Error | null"
    description="Error object if the status check failed"
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useSlackConnectionStatus` hook to check Slack connection status.

```tsx
import { useSlackConnectionStatus } from "@knocklabs/react";

const MyComponent = () => {
  const { isConnected, connectionStatus, error } = useSlackConnectionStatus();

  if (connectionStatus === "loading") return <div>Checking connection...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return (
    <div>
      <p>Slack Status: {isConnected ? "Connected" : "Not Connected"}</p>
      {isConnected && <p>You can receive Slack notifications</p>}
    </div>
  );
};
```
