The `useGuideContext` hook is used to access the guide context and client instance within a guide provider. This hook provides access to the guide client and theme information.

## Parameters

This hook does not accept any parameters.

## Returns

<Attributes>
  <Attribute
    name="client"
    type="KnockGuideClient"
    description="The guide client instance for managing guides"
  />
  <Attribute
    name="colorMode"
    type="'light' | 'dark'"
    description="The current theme color mode"
  />
</Attributes>

## Example

### Basic usage

The following example demonstrates how to use the `useGuideContext` hook to access the guide client and theme.

```tsx
import { useGuideContext } from "@knocklabs/react";

const MyGuideComponent = () => {
  const { client, colorMode } = useGuideContext();

  const dismissGuide = async (guideId: string) => {
    await client.dismiss(guideId);
  };

  return (
    <div className={`guide-container ${colorMode}`}>
      <p>Current theme: {colorMode}</p>
      <button onClick={() => dismissGuide("guide-123")}>Dismiss Guide</button>
    </div>
  );
};
```
