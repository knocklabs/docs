---
title: Building in-app UI in Javascript
---

The `@knocklabs/client` library is a low-level JavaScript SDK for interacting with Knock from the client side of your JS application.

**Note**: if you're already using React in your application, you might want to take a look at our [React library for building in-app UI](/in-app-ui/react/overview).

**Quick links**

- [`@knocklabs/client` on NPM](https://www.npmjs.com/package/@knocklabs/client)
- [Package on Github](https://github.com/knocklabs/knock-client-js)
- [Full reference guide](/in-app-ui/js/reference)
