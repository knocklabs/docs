Knock integrates with [Africa's Talking](https://africastalking.com/) to send SMS notifications to your recipients.

## Features

- Per environment configuration
- Sandbox mode

## Getting started

You can create a new Africa's Talking channel in the dashboard under the "Channels" section. From there, you'll need to configure the channel for each environment you have.

## Provider configuration

<Attributes>
  <Attribute
    name="API Key"
    type="string*"
    description="The API key from Africa's Talking."
  />
  <Attribute
    name="Username"
    type="string*"
    description="The username from Africa's Talking."
  />
  <Attribute
    name="Short code"
    type="string (optional)"
    description="The short code to send messages from."
  />
</Attributes>

## Recipient data requirements

In order to send an SMS notification you'll need a valid `phone_number` property set on your recipient.
